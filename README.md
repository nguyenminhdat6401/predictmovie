# 🚀 XGBoost Prediction Project

## 📌 Giới thiệu
Dự án sử dụng mô hình **XGBoost** đã được huấn luyện để dự báo dữ liệu đầu vào thông qua giao diện web.

## 🧠 Công nghệ
- Python
- XGBoost
- Pandas, NumPy
- Scikit-learn
- Flask / FastAPI
- HTML, CSS

## 📂 Cấu trúc thư mục
- `models/` : Lưu model đã train  
- `src/` : Code xử lý và dự đoán  
- `static/` : CSS, JS  
- `templates/` : Giao diện HTML  
- `requirements.txt` : Thư viện  

## ⚙️ Cài đặt & chạy
pip install -r requirements.txt  
python src/app.py  

👉 Truy cập: http://localhost:5000

## ⚡ Cách hoạt động
Input → Tiền xử lý → Load model → Dự đoán → Output

## 📊 Model
- Thuật toán: XGBoost (Regressor / Classifier)
- Metric: Accuracy / RMSE / MAE
